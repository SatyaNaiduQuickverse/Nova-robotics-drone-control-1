Quick reference:
THST_EXPO — Thrust curve. 0.65=linear, 0.85=gentle low-end. Raise for overpowered builds.
SPIN_MAX — Max motor output cap. 0.95=full. Set 0.75 to limit power by 25%.

Trim
When to use which
Airframe Trim (AHRS) — fixes the drone drifting in a direction during hover. Use this for weight imbalance (off-center battery, asymmetric payload) or a crooked FC mount. Active in every flight mode, including LOITER and autonomous. Typical range ±2°. Procedure: hover in ALT_HOLD with no wind, observe drift direction, dial in opposite trim, repeat.
TX Stick Trim (RCx_TRIM) — the PWM value the FC treats as "stick centered" for each channel. Set by Mission Planner radio calibration to whatever your TX actually outputs at rest (often not exactly 1500). Only active in manual stick modes. Don't use this to fix weight imbalance — it won't help in LOITER and will fight your calibration. Leave alone unless you know why you're changing it.
Rule: drift in LOITER → AHRS. Sticks feel off-center on the TX → recalibrate in Mission Planner, don't hand-edit RCx_TRIM.

Safety & Geofence
Configure flight-controller failsafes and the geofence boundary. Each section has its own Save button — changes are written to the FC immediately and persist across reboots. Click Load to refresh from the FC.
RC Link Failsafe
If the RC transmitter signal is lost (out of range, TX off, receiver dead), the FC will take the action below.
GCS Link Failsafe
Triggered when MAVLink heartbeats from the ground station stop arriving. Timeout = how long to wait before acting.
Battery Failsafe
Two-tier voltage monitoring. Low triggers a graceful response (e.g. RTL). Critical is last-ditch. Set either voltage to 0 to disable that tier. Rule of thumb: 3S pack ≈ 10.5 V / 10.0 V  ·  4S ≈ 14.0 V / 13.6 V  ·  6S ≈ 21.0 V / 20.4 V.
Voltage must stay below threshold for this duration before the action fires (avoids false trips on throttle sag).
Failsafe Behavior
Fine-tune how the failsafes above interact with the current flight mode.
Geofence
Invisible walls the drone can't cross. Combine a ceiling, floor, circle around home, and a custom polygon zone. Breach action fires the moment any enabled boundary is crossed.

Polygon points
One lat, lon pair per line in decimal degrees. Need at least 3 points. Don't repeat the first point — the FC closes the loop automatically. Uploading replaces any existing polygon.
Calibration
⚠ SAFETY: All calibrations require the drone to be disarmed. For Motor Test, remove propellers — the FC will spin motors on command. Stay clear of the frame.
Motor Test
Spin individual motors at a set throttle to verify wiring, direction, and order. Props must be off.
